# Changelog

All notable changes to UXistentialism Studio are recorded here. The format
follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/). Because
continuity is a core value of this product, its own history is kept
deliberately.

## [Unreleased]

### Added
- Foundation documents establishing shared direction before any application
  code: `PRODUCT.md`, `DESIGN.md`, `ARCHITECTURE.md`, `ROADMAP.md`,
  `CHANGELOG.md`, and `CONTRIBUTING.md`.
- Project overview and philosophy in `README.md`.
- An explicit product ontology — **Idea** as the primary object, first-class
  **Connections**, append-only **Lineage**, and the six spaces framed as
  **modes** and relationships to time rather than pipeline stages.
