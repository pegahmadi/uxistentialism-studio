# UXistentialism Studio

UXistentialism Studio is an operating system for intellectual work.

Rather than helping people write faster, it helps ideas mature.

The product supports the complete lifecycle of public thinking:

→ Today
→ Field
→ Formation
→ Iteration
→ Distribution
→ Memory

Each space answers a different question:

- **Today** — Where should I spend my attention?
- **Field** — What is happening in the world?
- **Formation** — What is worth writing?
- **Iteration** — How do I strengthen this work?
- **Distribution** — How should this idea travel?
- **Memory** — What has this work become?

The goal is not to optimize content production.

The goal is to build an environment where ideas accumulate, connect, mature, and develop into enduring intellectual infrastructure over time.

This repository contains the evolving design, architecture, and implementation of UXistentialism Studio.
