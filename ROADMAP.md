# Roadmap

This roadmap is directional, not a schedule. It sequences the work so that the
product's core belief — that ideas mature through connection and history — is
proven before anything is scaled or polished. Phases are ordered by dependency.

## Phase 0 — Foundation _(current)_

Establish shared understanding before writing application code.

- [x] Project overview and philosophy (`README.md`)
- [x] Foundation documents (`PRODUCT`, `DESIGN`, `ARCHITECTURE`, `ROADMAP`,
  `CHANGELOG`, `CONTRIBUTING`)
- [ ] Decide the initial technical stack and scaffold the application shell

## Phase 1 — The Spine

Make the domain real with the minimum needed to develop one idea.

- [ ] Define the core **Idea** model with append-only lineage
- [ ] Make **Connections** first-class from the start (ideas to ideas,
  observations, sources, discussions)
- [ ] Build **Formation** — capturing an idea and shaping it into a thesis
- [ ] Build **Iteration** — revisiting and strengthening an idea, preserving
  its history

## Phase 2 — Input and Output

Connect the domain to the world.

- [ ] Build **Field** — capturing observations and sources as they stream in
- [ ] Build **Distribution** — shaping how an idea travels
- [ ] Connect signals and sources to the ideas they inform

## Phase 3 — Continuity

Make the body of work cumulative and legible.

- [ ] Build **Memory** — what an idea has become over time
- [ ] A graph interface over the connections that already exist in the model
- [ ] Build **Today** — orientation drawn from the state of everything else

## Phase 4 — Depth

Deepen the environment once the loop is complete.

- [ ] Concrete visual language and design system
- [ ] Persistence, sync, and durability
- [ ] Refinement of each mode's distinct posture and relationship to time

_Later phases are intentionally sketched. They will be specified as the
earlier phases teach us what the product actually needs. Note that connections
and lineage appear in Phase 1 by design — they are part of the model, not
features added at the end._
